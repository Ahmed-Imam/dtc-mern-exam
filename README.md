# EXAM

## Installation

```bash
# clone the repo
$ git clone https://github.com/Ahmed-Imam/dtc-mern-exam.git

# go into app directory
$ cd dtc-mern-exam

# install app dependencies
$ npm i or yarn install
# install app client dependencies
$ cd client 
$ npm install 

# run the app in main dir 
$ cd dtc-mern-exam
$ npm run dev

```
